[Slides Studio](https://uuoocl.github.io/slides-studio/) is an HTML page that enables Reveal.js presentations to control Open Broadcast Studio (OBS).  Reveal.js is an open source HTML presentation framework. It's a tool that enables anyone with a web browser to create fully-featured and beautiful presentations for free.

Create slides in HTML, Markdown or with the reveal.js  GUI Slides.com

![Drawing slides studio diagram.excalidraw.png](/img/user/Excalidraw/Drawing%20slides%20studio%20diagram.excalidraw.png)

# Requirements
To use Slides Studio the following pieces are required. 
1. Open Broadcast Studio(OBS)
	- OBS is a free and Open Source software for video recording and live streaming.
	- [Download from obsproject.com](https://obsproject.com/)   

2. A [Reveal js]() slide deck
	-  example slides.com deck, [https://slides.com/uuoocl/deck-48ea10](https://slides.com/uuoocl/deck-48ea10)

>[!Note]
>Optionally, create a Slides.com account (starting at $5 usd per month) to create your own presentations. 

3. An HTTP server
	- Slides Studio is a webpage hosted on Github pages. 
		- [slides-studio hosted on Github pages](https://uuoocl.github.io/slides-studio/)
	- Download the repo and run it on a localhost.  
		- [Obsidian plugin slides extended](https://www.ebullient.dev/projects/obsidian-slides-extended/)
		- [Simple OBS Python Script HTTP Server](https://github.com/UUoocl/python-http-server)

# Configure OBS: 
## Import the Slide Studio Scene Collection

1. Download the Slides Studio OBS Scene Collection [download link.](https://github.com/UUoocl/slides-studio/blob/main/obs_Collections/Slidescom_Collection.json) 
2. In OBS Import the Slides Studio OBS collection
	a.  On the file menu click, "Scene Collection" -> "Import"
	![Pasted image 20250802193752.png](/img/user/media/Pasted%20image%2020250802193752.png) 
3. Select the Scene Collection.
	a.   "Scene Collection" -> "Slides.com_Collection"

## Enable the OBS WebSocket Server

1. On the file menu click, "Tools" -> "WebSocket Server Settings"
2. Check the "Enable WebSocket server" option
![Pasted image 20250802195254.png](/img/user/media/Pasted%20image%2020250802195254.png)
3. Click the "Show Connect Info" button to view the Server Password.

## Select a Camera
1. Select the Scene named "Set Camera"
2. Double click the Source named "Camera 1" to open the source properties window
3. Use the "Device" drop down to select a camera. 
![Drawing Set Camera.excalidraw.png](/img/user/Excalidraw/Drawing%20Set%20Camera.excalidraw.png)

# Add Slide Control Tags
## Choose the "Studio" layout

After connecting to the OBS WebSocket server, the Slide Studio view will appear. 
1. Use the "Layout" button and select the "Studio" option. 

2. For each slide, click the cell to choose an available option.  
	  - Slide Position: where the slides appears on the scene
	  - camera Position: where the Camera appears on the scene
	  - Camera Shape: change the camera from a rectangle to a different shape
	  - Scene: OBS scenes that begin with "Scene"
![Drawing add control tags.excalidraw.png](/img/user/Excalidraw/Drawing%20add%20control%20tags.excalidraw.png)

## Navigating Slides
## 1. Current Slide navigation
Click the ">" button to advance to the next slide, or the "<" button to return to the previous slide. 

The keyboard keys left and right arrow keys can navigate the slides forward and back. 

## 2. Slide tags navigation
Click on the slide number in the "Index" column to jump to a specific slide. 

Press the keyboard "spacebar" key to advance to the next slide.

Click the "Layout" button, then select "Teleprompter".

The teleprompter view can set the font size, scroll speed, flip text and play/pause
