# Setup

Open the `open-obs.html` in a Chrome browser.

## Configure OBS and the WebSocket Server

> [!note] OBS Settings
 > **Configuration name**:  The settings will be saved to the browser local storage. Enter a name to retrieve the settings later.  
>
 > **App Name**: The default name for OBS in macOS is "OBS".  If another name has been assigned, enter that name. For example if testing a new version a User may have set the name to "OBS40"
 >
 >**Profile**: Enter the name of the OBS Profile to load.
 > **Collection**: Enter the name of the OBS Collection to load.
 >
 > **Port**: Enter the WebSocket Server Port.  The default port is 4455
 > **Password**: Enter the WebSocket Server Password.
 > 
 > **Debug Mode**: Enable the Browser Source debug mode.  To debug a browser source open Chrome to "http://localhost:9222/"
 
## Open OBS with macOS Shortcuts
If the macOS "Open OBS" Shortcuts have been imported, then press the "Open OBS" button. 

## Open OBS with a Command Line 
Press the "Copy Command" button.
Open the "Terminal App" in macOS or "Command Line" in  Windows

Paste the open OBS command.
